# Rick and Morty Explorer | AI-Driven Development Study

This repository is a technical experiment focused on **AI-driven software engineering**. The entire application, from architecture to UI components, was developed using advanced AI tools like **Antigravity** and **OpenCode** to explore the current capabilities and limitations of LLMs in building modern React applications.
